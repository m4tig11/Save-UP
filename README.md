Save-UP

Descripción

Save-UP es una aplicación de escritorio desarrollada en Python con Tkinter que permite a los usuarios gestionar sus gastos de manera eficiente. La aplicación ofrece una interfaz gráfica amigable donde los usuarios pueden registrar, visualizar y analizar sus gastos en diferentes categorías.

Características

Registro de gastos: Permite ingresar gastos en múltiples categorías.

Visualización gráfica: Muestra un gráfico de pastel con los gastos por categoría.

Historial de gastos: Consulta y seguimiento de los gastos realizados.

Gestión de saldo: Actualiza el saldo del usuario al agregar gastos.

Interfaz intuitiva: Diseño amigable con botones para cada categoría.

Tecnologías Utilizadas

Python (Tkinter para la interfaz gráfica)

SQLite3 (Base de datos para almacenamiento de gastos y usuarios)

Matplotlib (Visualización de datos mediante gráficos)

Uso

Inicia sesión o regístrate.

Ingresa el monto del gasto y selecciona una categoría.

Visualiza el gráfico actualizado con los gastos del mes.

Consulta el historial de gastos cuando lo necesites.
